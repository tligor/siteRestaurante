## Site Restaurante

![image](https://github.com/tligor/siteRestaurante/blob/main/Restaurante.png)

Um site de apresentação de restaurante com seu cardápio.

### Funcionalidades

- **Apresentação do Restaurante**: Página inicial com informações sobre o restaurante, incluindo horário de funcionamento, contato e pratos ofertados.
- **Cardápio**: Página dedicada ao cardápio com categorias de pratos e descrições.
- **Integração com Google APIs**: Utilização das APIs do Google para inclusão de fontes e outros recursos.

### Tecnologias Utilizadas

- **HTML**: Estruturação do conteúdo do site.
- **JavaScript**: Implementação da lógica de interação, slider e dinamismo do site.
- **CSS**: Estilização do site para torná-lo visualmente atraente.
- **Bootstrap**: Framework CSS para design responsivo e componentes prontos.
- **Google APIs**: Integração com serviços do Google, como fontes.
